🔗 iOS 快捷指令设置

📸 方式一：截图识别记账（推荐）

1. 在 iPhone 上打开「快捷指令」App

2. 创建新快捷指令，添加以下操作：

　①「获取最新截图」（从相册取最新截图）

　②「分享」→ 选择「小杨专属记账」

3. 保存快捷指令，命名为「截图记账」

4. 在 iPhone 设置 → Action Button 或轻点背面 → 绑定此快捷指令

💡 使用：支付后截图 → 按Action Button/轻点背面 → 自动识别金额！

⚡ 方式二：快速输入记账

1. 创建新快捷指令

2. 添加「要求输入」→ 提示"输入金额"

3. 添加「打开 URL」操作

4. URL 格式：

https://你的地址/index.html?amount=25&type=expense¬e=午饭&quick=1

🎯 方式三：Share Sheet 直接分享

1. 在任何 App 中截图或查看支付截图

2. 点击分享按钮 → 选择「小杨专属记账」

3. 自动跳转到App进行OCR识别

参数说明：

• amount = 金额

• type = expense(支出) / income(收入) / debt(欠款)

• category = 分类ID（可选，自动识别）

• note = 备注（可选）

• account = personal / company（可选）

• quick = 1 自动保存（可选）
